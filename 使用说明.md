# alipay 
# 环境依

PHP5.0以上，且需要开启CURL服务、SSL服务。

# 文件对应说明

* pc.php        电脑网站支付
* wap.php       手机网站支付
* qrcode.php    当面付【扫码支付】
* barcode.php   当面付【条码支付】
* refund.php	退款

---
* authorize.php  网页授权获取用户信息


# 注意事项

1.需要用到支付宝哪一种支付方式，就只下载对应的单个文件即可。

2.文件开头的配置信息必须完善
